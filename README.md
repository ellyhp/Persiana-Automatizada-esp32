# Persiana-Automatizada-esp32
Proyecto de persiana inteligente automatizada basado en ESP32 y tecnología IoT, con control local y remoto mediante WiFi y Blynk. El sistema permite abrir, cerrar y detener la persiana, configurar límites de recorrido y almacenar parámetros, mejorando la comodidad, automatización y eficiencia del usuario. en entornos domésticos modernos. útil.
